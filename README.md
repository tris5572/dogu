# dogu (道具)

dogu は、Web ブラウザで使えるちょっとした便利道具（ツール）を集めたものです。すべての動作は Web ブラウザ内で完結します。

> [!IMPORTANT]
> 🚧 まだ作成中であり、機能はほとんどありません。

## モチベーション

便利ツールを公開している Web サイトはいくつもありますが、サーバーへデータを送信する仕様だったり、デザインが野暮ったかったり、バグがあったり、ブラウザ上で処理が完結すると謳っていながらクローズドソースであるために真偽が疑わしい、というように、何かしらの引っ掛かりがあるものばかりだったので、自分で作りました。

## 使用技術

- TypeScript
- React 19 (SPA)
- CSS Modules
